MRH_GetLocalStreamLibraryVersionMinor
=====================================
The MRH_GetLocalStreamLibraryVersionMinor function is used to obtain 
the minor library version.

Header
------
The function definition can be found in the following file:

.. code-block:: c

    #include <libmrhls/MRH_LocalStreamRevision.h>


Syntax
------
.. code-block:: c

    int MRH_GetLocalStreamLibraryVersionMinor(void);


Parameters
----------
None.

Return Value
------------
The minor library version defined by 
:doc:`../../../Macros/libmrhls_Directory/MRH_LocalStreamRevision/MRH_LOCAL_STREAM_LIB_VERSION_MINOR`.

Remarks
-------
None.
